# TranslEYEtor
An automatic AI powered screen translator.

## Current Version
V0.2 - Alpha <br>
- Translated text is now visualized on screen around the chosen translation area.
- App now attempts to cache model weights and load them locally.
- Model is now cached near the python script in .model_cache.

## Version History
V0.1 - Alpha <br>
- Added basic 480p screen capture on CTRL press.

## System Requirements
- An OS
- Any CPU
- Integrated graphics

## Required Dependencies
- Python 3.13 (on PATH)
- Pip (on PATH)

## How To Use
1. Download and run the python script
2. Wait for the necessary dependencies to be installed
3. When the program is ready, press CTRL to capture a 480P image at the center of your cursor
4. The text in the image will be translated to English
